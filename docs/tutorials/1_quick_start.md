## 1. Подготовка окружения и установка

Прежде чем начать общаться с математическим гением, убедитесь, что у вас установлен **Python 3.9 или выше**. Проект использует фреймворк PyTorch для нейросетей и библиотеку Rich для красивого терминального интерфейса.

Откройте терминал (командную строку) и выполните:

```bash
# 1. Установка базового фреймворка (PyTorch) для вычислений
pip install torch torchvision torchaudio

# 2. Установка интерфейсных и системных утилит
pip install rich psutil matplotlib
```

> [!NOTE]
> **Владельцам видеокарт Intel (Arc / Iris Xe):** Pythagoras 1.0 "из коробки" поддерживает аппаратное ускорение XPU. Если вы хотите, чтобы модель работала и обучалась максимально быстро на железе Intel, установите специальное расширение:
> `python -m pip install intel-extension-for-pytorch`

## 2. Запуск чата и интерфейсы

В корне проекта вы найдете несколько способов пообщаться с моделью. Мы написали их под разные нужды:

*   `chat_math.py` — **Минималистичная версия.** Быстрый запуск, ничего лишнего. Подходит для серверов без поддержки красивой отрисовки.
*   `chat_math_rich.py` — **Rich-версия.** Отрисовывает красивые панели, прогресс-бары и цветной текст.
*   `pythagoras_hub.py` — **Универсальный центр управления (Рекомендуется!).** Это главная "база" проекта, откуда можно запустить и чат, и тренировку, и диагностику.

**Запустим центр управления:**
```bash
python pythagoras_hub.py
```

Перед вами появится интерактивное главное меню. Выберите пункт **«2. ЧАТ С МОДЕЛЬЮ»**, введя цифру `2` и нажав `Enter`. Скрипт загрузит веса из файла `math_model_weights.pth` (если он есть) и подготовит нейросеть к диалогу.

## 3. Как правильно задавать вопросы (Формат ввода)

Модель **Pythagoras 1.0** — это посимвольный Трансформер. Она обучалась на строгом, "чистом" математическом формате данных без пробелов и слов. Поэтому примеры нужно вводить в том же формате, в каком она их видела на этапе тренировки.

**✅ Правильный формат ввода:**
```text
123+456=
99-15=
```
*(Строго без пробелов, только числа, знак операции посередине и обязательный знак равно на конце)*

**❌ Неправильный формат ввода:**
```text
123 + 456 =         # Ошибка: модель не знает, что такое пробел
Сколько будет 123+456? # Ошибка: в словаре модели нет букв русского языка
123+456             # Ошибка: модель ждет знака '=', чтобы начать генерировать ответ
```

Сразу после того, как вы введете знак `=` и нажмете Enter, модель подхватит ваш текст и начнет генерировать ответ, посимвольно (цифра за цифрой) выводя результат на экран!

## 4. Что происходит под капотом (Токенизация и Генерация)

Когда вы нажимаете `Enter` и отправляете `123+456=`, происходит настоящая математическая магия:

1.  **Токенизация (Character-level)**: Ваш текст разбивается не на слова, а на отдельные символы: `['1', '2', '3', '+', '4', '5', '6', '=']`. Каждый символ превращается в числовой индекс из словаря (сохраненного в `math_vocab.pkl`).
2.  **Эмбеддинг (Превращение в смысл)**: Каждый индекс переводится в вектор из 256 чисел, плюс добавляется информация о позиции (чтобы модель понимала, где единицы, а где сотни).
3.  **Глубокая обработка (Transformer Blocks)**: Векторы проходят через 6 слоев (блоков Трансформера), где механизм Многоголового Внимания (Multi-Head Attention) ищет связи (например, "тройка должна сложиться с шестеркой").
4.  **Предсказание (LM Head)**: Финальный слой предсказывает вероятности для следующего символа. Модель выбирает наиболее вероятный (например, `5`).
5.  **Авторегрессия**: Этот предсказанный символ `5` добавляется к исходной строке (`123+456=5`). Весь этот длинный процесс повторяется заново для предсказания следующей цифры `7`, потом `9`. Процесс останавливается только тогда, когда модель предсказывает специальный скрытый символ `\n` (конец примера).

## 5. Глоссарий Быстрого Старта

| Термин | Простое и понятное объяснение |
| :--- | :--- |
| **Веса модели (`.pth`)** | Сохраненная "память" нейросети. Файл с матрицами чисел, в которых зашифрованы все математические правила, выученные во время тренировки. |
| **Токенизация** | Процесс нарезки текста на кусочки, понятные нейросети. Pythagoras режет по одной букве (посимвольно). |
| **Авторегрессия** | Способность ИИ использовать собственные только что сгенерированные ответы как часть контекста для генерации продолжения. |
| **XPU / CUDA** | Технологии, позволяющие перенести вычисления с процессора (CPU) на видеокарту для гигантского ускорения работы ИИ. |
